Fruit Rage

Android Game

Fruit Rage is an Android game similar to Candy Crush, with the interesting twist that you play against an AI, as compared to most variants where you aim to outscore yourself.

The AI is a Minimax agent with Alpha-Beta pruning. It dynamically selects a lookahead depending on its remaining team. The objective of the game is to score more points than the AI before the board runs out of fruits.

This project was co-developed with Rohit Sahasrabuddhe, my classmate at the University of Southern California. I additionally designed custom 2D art assets for this game in Adobe Photoshop.

(The original command-line Fruit Rage game was developed as an assignment for the Artificial Intelligence course at USC, later reinvented by us as a fun visual game.)

  • Date: October 2017 - January 2018
  • Links: GitHub repository

DbTrack

Employee Tracking System in ASP.NET/C#

DbTrack was developed by students of K. J. Somaiya College of Engineering (Hemang Tailor, Chintan Doshi and Akshat Gandhi) to help officers of the Mumbai Anti-Corruption Bureau (ACB) to manage their employees.

In 2015, the ACB needed updates to the system with additional features. I revised the login module and storage mechanisms, implementing additional security features such as CAPTCHA, password hashing and a password reset function.

My responsibilities also included providing technical support using TeamViewer to solve configuration and network issues, and deploying the updated software on the ACB computers.

In a team of 6 (including myself - Siddhesh Karekar, Bhushan Pathak, Ashutosh Mahajan, Amogh Bhabal, Aditya Godambe, and Chinmay Karanjkar) we conducted training sessions for the police to educate them on the software's usage.

K. J. Somaiya College of Engineering awarded us with a Letter of Appreciation for our efforts.

  • Date: May 2015 - February 2016, including development and maintenance
  • Client: Anti-Corruption Bureau, Mumbai, India
  • Links: Letter of Appreciation, Anti-Corruption Bureau website

Deimos

A data mining/machine learning in Java with a JavaFX GUI

For any organization that serves customers, profiling their users is important, so as to target them with relevant information.

Deimos is a Java-based system that aims to help doing the same. It estimates the user's demographic attributes - specifically, their age, gender and interests, by analyzing web pages they have visited.

It uses a TF-IDF classifier to analyize browser history, which is later fed into a neural network to generate output. The system allows visualizing output in the form of a nice tag cloud:

During our tests, the system was applied to shopping website data to profile viewers based on visited web pages, and performed with reasonable accuracy. Check out the source code and the research we've published below.

  • Client: K. J. Somaiya College of Engineering (final year project)
  • Team: Siddhesh Karekar (leader), Bhushan Pathak, Amogh Bhabal
  • Links: GitHub repository, Research Paper at IJCA

Dots

Java Game with Swing GUI

Dots is an attempt to recreate the pencil-and-paper children's game of Dots and Boxes.

The game features a Swing GUI, but is also fully functional via the console. The GUI allows you to customize the number of players, each player getting their own unique color.

Rules are the same: Players take turns joining two horizontally or vertically adjacent dots by a line. When a player completes a 1×1 box, they 'win' that square and play again. The player who has colored more boxes wins when all boxes are colored.

  • Date: September 2014 - October 2014
  • Client: K. J. Somaiya College of Engineering (coursework for Object-Oriented Programming)
  • Links: GitHub repository
