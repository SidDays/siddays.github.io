About me

Hi! I'm a grad student pursuing my MS in Computer Science at the University of Southern California. I'm looking for a Summer 2018 Software Engineering internship to utilize my skills and make a valuable contribution, and also develop my own abilities in Computer Science.

Software development is my passion. I have developed and worked on several projects: - Fruit Rage, an android game similar to Bejeweled/Candy Crush but you play against an AI - Deimos, a Java data mining and machine learning system to predict user demographics - DbTrack, an ASP .NET/C# web-based employee tracking system - Dots, a Java Swing game recreating the pen-and-paper 'dots and boxes' game

My skills include Java, Android, SQL, HTML, CSS, and JavaScript, the Microsoft Office suite, Adobe Photoshop, Sony Vegas Pro and FL Studio.
